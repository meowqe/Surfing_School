✅ Готовый шаблон User Story Map (USM) для твоего проекта OLIMP.RU в Figma
Я проанализировал оба скриншота (лендинг + модальное окно входа) и сделал полноценную, детальную карту пользовательских историй специально под твой дизайн сайта.
Как использовать шаблон в Figma (2 минуты):
	1	Создай новый фрейм 3840×2000 px (или больше).
	2	Сверху — заголовок «User Story Map • OLIMP.RU».
	3	Нарисуй 5 больших колонок (горизонтальная ось — этапы пути пользователя).
	4	В каждой колонке — строки-стикеры (прямоугольники 320×120 px, закруглённые, разные цвета по приоритету).
	5	Цвета стикеров (рекомендую):
	◦	Синий — must-have (то, что уже есть на скринах)
	◦	Зелёный — next (логично добавить в ближайшее время)
	◦	Жёлтый — later / nice-to-have
	6	Скопируй текст сторис ниже прямо в Figma (каждый пункт — отдельный стикер).
Персоны (сверху над картой)
	•	Гость (незарегистрированный посетитель)
	•	Родитель (главный покупатель подписки)
	•	Ребёнок / Школьник (конечный пользователь обучения)
	•	Учитель (второстепенная аудитория)
